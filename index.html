Analizador Inteligente de Funciones

Para funciones a trozos, usa el formato: `x < 0 : -x ; x >= 0 : x^2`.
Puedes usar funciones como `sin(x)`, `cos(x)`, `log(x)`, `abs(x)`, `exp(x)`, `sqrt(x)` etc.

Cómo ingresar problemas:

  • Funcion lineal:Suma/Resta: `x + 2`, `3x - 5`
  • Multiplicación: `2 * x`, `3x` (se asume 3*x), `(x+1)(x-2)`
  • Funcion racinal: `1 / (x - 1)`
  • Funcion cuadratica y cubica: `x^2`, `x^3`, `sqrt(x)` Funcion raíz cuadrada
  • Funcion Constantes: `pi` (para $\\pi$), `e` (para el número de Euler)
  • Paréntesis: Usa paréntesis para agrupar operaciones, ejemplo: `(x + 1) / (x - 1)`
Introduce un número para la cantidad de puntos a graficar. Prueba con `10`, `20` o más para ver la forma de la función. Un valor de `0` (cero) solo mostrará la línea continua.
